    Used Tech

  • Crawling

    • Python
    • Multi Thread

    Coverletter 데이터 수집에 사용된 crawling 기술입니다.

    대량의 데이터를 Crawling 하면서 시간단축을 위해 Thread 기법을 사용하였습니다.

    다중 for문이 사용되는 구간에 다중 thread를 접목해서 시간을 단축시켰습니다.

    Multi Thread py script github

  • Auto Labeling

    • Python
    • Neural Network
    • Multi label Classification

    Auto Labeling은 레이블이 되어있지 않은 데이터에 레이블링을 해주는 기법입니다.

    소수의 데이터에 직접 레이블링을 해주고 기본 Multi Layer Perceptron Model을 만들어서 다수의 데이터를 학습시켰습니다.

    이 과정에서 하나의 레이블이 아닌 복수의 레이블을 할당할 수 있는 Multi Label Classification을 사용하였습니다.

    Multi Label github

  • Text Generation

    • Long Short-Term Memory
    • Sparse Cross Entropy

    Coverletter 작성 중, 자주 사용되는 조사, 관용어구를 자동완성 시켜주는 기능입니다.

    LSTM을 사용하여 sequence text data 처리성능을 향상시켰습니다.

    Out-Of-Memory 문제에 대해 Sparse Cross Entropy를 사용하여 memory use를 감소시켰습니다.

    Text Generation Model github

  • Summerization

    • Sequence To Sequence
    • Hangul Romanize

    Coverletter를 한 문장으로 표현할 수 있도록 요약하는 기능입니다.

    한글은 글자의 다양성으로 인해 Out-Of-Memory에 자주 직면합니다. 이를 해결하기 위해 한글 Romanize 기능을 사용하여 memory use를 감소시켰습니다.

    Summarization Model github
